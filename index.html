Younus Scientific Calculator Guide

Basic Operations: +, −, ×, ÷

sin(x), cos(x), tan(x): Trigonometric functions in degrees.

log(x): Base-10 logarithm. Example: log(100)=2

ln(x): Natural logarithm (base e)

√: Square root. Example: √25 = 5

x² / x³: Power function. Example: Math.pow(5,2) = 25

π: Pi constant ≈ 3.14159

e: Euler number ≈ 2.718

( ): Use parentheses for order of operations.

%: Percentage.

DEL: Delete last character.

C: Clear display.

=: Calculate result.

All functions work offline and are mobile-optimized!
